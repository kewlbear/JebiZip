# JebiZip

A minimal Swift package for ZIP file format.

## Usage

```
import JebiZip

try unzip(zipURL, to: directoryURL)
```

## Requirements

- iOS 9+ / macOS 10.11+
- Xcode 11+
- Swift 5.1+

## Installation

### Xcode 11

File > Swift Packages > Add Package Dependency...
